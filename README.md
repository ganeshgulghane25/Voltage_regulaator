Linear & Variable Voltage Regulator (7805 + LM317)

This project demonstrates the design and implementation of two power supply circuits: a fixed 5V linear voltage regulator using the 7805 IC, and a variable voltage regulator using the LM317 IC. Both Proteus simulations and real hardware builds are included to show the complete workflow from circuit design to practical testing.
